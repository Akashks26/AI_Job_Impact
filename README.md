# 🔮 AI Job Impact & Career Intelligence Advisor (2030)

## 🚀 Overview

The **AI Job Advisor 2030** is a data-driven application built using **Streamlit, Machine Learning, and NLP** that helps users:

- 🔍 Explore job roles and domains
- 🤖 Understand AI impact on careers
- 📊 Analyze automation risk & survival score
- 💰 View salary predictions (multi-currency)
- 🧠 Get AI-assisted task insights
- 🛠 Follow a structured career roadmap

---

## 🧠 Key Features

### 🎯 Smart Job Matching
- Uses **Sentence Transformers (NLP embeddings)**
- Finds closest job role intelligently

### 🤖 AI Impact Analysis
- Automation Probability (2030)
- AI Assist Boost
- Tech Growth Factor

### 💰 Salary Forecasting
- 5-year projected salary growth
- Multi-currency support (INR, USD, EUR, GBP)

### 🛡 AI Survival Score
- Combines automation risk + growth potential
- Helps understand long-term job stability

### 📊 Interactive Visualizations
- Salary trend charts
- Skill radar charts
- Career growth progression

### 🔥 AI-Assisted Task Insights
- Shows how AI will help in your job role
- Dynamic task generation for all roles

### 🛠 Career Roadmap Generator
- Foundation → Core → Advanced skills
- Industry-ready preparation path

## ⚙️ Installation & Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Akashks26/AI_Job_Impact.git
cd AI_Job_Impact
```

### 2️⃣ Create Virtual Environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python -m venv venv
source venv/bin/activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
streamlit run app.py
```
